5Intel 主機板常見 Q-CODE 參考指南

Q-CODE狀態 / 故障說明常見除錯方向
00CPU 錯誤 / 未偵測到 CPU檢查 CPU 是否正確安裝、底部接點有無髒汙或斷針、EPS 12V 供電是否接上、CPU 供電模組是否短路。
53 / 55記憶體初始化錯誤重插記憶體、使用橡皮擦清理金手指、測試單根記憶體。若仍報錯，需檢查 CPU Socket 腳座針腳是否彎曲變形。
62PCH 執行階段服務安裝通常與 PCIe 裝置或顯示卡(VGA)有關。請檢查 PCIe 插槽接觸狀態、顯示卡輔助供電是否正常，或重新插拔顯示卡。
99Super IO 初始化常見於 I/O 周邊卡住。檢查主機板上的風扇接頭、前置 USB 設備、鍵盤滑鼠，或是 PCIe 擴充卡是否短路或異常。
A0 / A2IDE / SATA 初始化 (硬碟偵測)顯示 A0 通常代表正常進入系統。若卡在 A2，則需檢查 SATA 排線、M.2 硬碟接觸、SATA Controller 或背板是否異常。
b4USB 設備熱插拔移除所有 USB 外部設備再開機測試，並用手電筒檢查主機板與機殼的 USB 接口是否出現金屬接腳變形短路。
d6未偵測到 VGA 輸出設備無顯示畫面，系統找不到顯示晶片。檢查獨立顯示卡安裝、螢幕訊號線，或先拔除獨顯切換至內顯進行交叉測試。

* 此表為常見代碼參考，實際定義可能因不同主機板廠牌與 BIOS 版本而略有差異。
